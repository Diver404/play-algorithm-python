## 声明

本项目旨在用最直观、简练的方式讲解数据结构和算法，所有实现参考了很多书籍或前辈的分享，只用于**学习和演示**，请注意以下几点：

* 我们并不完全保证内容的标准性，如遇到有问题的地方还请及时指正
* 其中很多内容可能来自书籍或网络上的分享，如果冒犯了您的著作权等，请及时指出，我们会第一时间处理
* 所有实现均用于学习和演示目的，它们的效率将低于很多标准库中的实现
* 无特殊说明，代码部分均基于`python3`
